# ATLAS.002 — Dropping bounded-degree breaks FO^k-local control

- Target statement: LG.HOOK.Locality
- Dropped assumption: bounded degree (Δ < ∞)
- Witness kind: unbounded-degree graph family
- Minimal description:
  - Unbounded-degree permits radius-R neighborhoods whose FO^k-types encode arbitrarily large global distinctions via high-arity adjacency patterns within fixed radius.
  - Therefore any locality-to-patch typing bound that is uniform in Δ fails.
- Verification method:
  - Necessity witness: any uniform finite-type bound requires Δ-boundedness.
