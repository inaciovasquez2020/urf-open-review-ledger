\begin{abstract}
We study the expressive limits of locality in finite-variable logic on
bounded-degree graphs. We show that FO$^k$-local refinement procedures admit
only a finite per-step outcome alphabet, yielding an a priori bound on
information extracted at each local query. Building on this, we introduce a
configuration pumping principle based on finite FO$^k$ type counting and prove
a conditional closure of the FO$^k$ locality wall: sufficiently large local
cycle complexity forces FO$^k$ type splitting within bounded radius. Our
approach avoids global linkage, holonomy, or Ehrenfeucht--Fraïssé cycle-trapping
arguments and relies solely on bounded degree, bounded quantifier rank, and
local configuration repetition. All results are stated with explicit
conditional dependencies.
\end{abstract}
