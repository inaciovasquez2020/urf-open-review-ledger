# WL / Obstruction Glossary

**CFI Construction**  
A standard family of graphs/formulas designed to fool low-dimensional distinguishers (e.g., WL-k).

**Dimension (WL-k)**  
The k in the k-dimensional Weisfeiler–Leman refinement.

**Distinguishability**  
A property that two objects yield different invariants under a specified procedure (WL, logic, spectrum, etc.).

**Graph Refinement Profile**  
The sequence of partitions / colorings produced by iterative refinement.

**Hierarchy of Distinguishers**  
An ordering of methods by strength (e.g., WL-1 < WL-2 < …, FO² < FO³ < …).

**Local Equivalence Class (WL-k)**  
The class of vertices/tuples sharing the same WL-k color after stabilization.

**Refinement Equivalence**  
Two structures are equivalent if the chosen refinement stabilizes to the same invariant.

**WL Invariant**  
Any value/function constant on WL-k equivalence classes (colors, stabilized multisets, counts).

**WL Obstruction**  
A certified instance where a WL-k procedure provably cannot distinguish required structure.
