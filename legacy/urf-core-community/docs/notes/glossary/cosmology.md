# Cosmology (DFM–MKC) Glossary

**DFM (Dark Fluid Model)**  
A cosmological model using an effective dark-sector fluid description.

**MKC (Minimal Kinetic Coupling)**  
A specific kinetic-coupling mechanism defining interaction terms in the model.

**Constraint Surface**  
The set of parameter values consistent with a chosen dataset/likelihood and priors.

**Tension (H₀, S₈)**  
A discrepancy between inferred parameter values from distinct datasets or analyses.

**Model Plausibility Gate**  
A declared acceptance criterion (fit quality, stability, priors, diagnostics), if specified.
