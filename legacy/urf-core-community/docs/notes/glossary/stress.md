# Stress / Harness Glossary

**CI Smoke Invariant**  
A minimal invariant checked in CI to ensure the repo remains executable and structurally consistent.

**Deterministic Stress Scenario**  
A stress configuration with fixed seed/inputs so results are reproducible.

**Stress Counters**  
Measured quantities used to track failure modes or invariant violations under stress.

**Stress Test Harness**  
The code and configuration used to apply adversarial or boundary-condition tests.

**Verification Stress Index**  
A summary statistic of stress outcomes (pass/fail + measured margins), if defined.
