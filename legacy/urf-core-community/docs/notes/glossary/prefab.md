# Prefab / Packaging Glossary

**CI Invariant**  
A repository invariant enforced by CI (build determinism, schema validity, import rules).

**Deterministic Packager**  
A build/packaging process that produces identical outputs from identical inputs.

**Prefab Module**  
A packaged unit exporting executable realizations of Core semantics.

**Prefab Interface Contract**  
The constraints a prefab artifact must satisfy to be a valid downstream consumer.

**Semantic Import Constraint**  
A rule that downstream layers may import semantics but must not redefine meanings.
