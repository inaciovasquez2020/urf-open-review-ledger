# Cross-Repo Meta Glossary

**Canonical**  
Designated as the semantic reference state.

**Frozen**  
Declared stable; changes are restricted and must preserve meaning.

**Layer Stack**  
Core → Prefab → Infrastructure → Applications.

**Non-Scope**  
Explicitly excluded responsibilities of a repository/layer.

**Semantic Authority**  
The layer/repo that defines meanings and validity conditions.

**Structural Witness**  
An artifact exhibiting a failure mode or obstruction under stated rules.
