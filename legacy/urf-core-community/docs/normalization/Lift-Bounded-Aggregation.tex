\section{Beyond AC$^0$: Bounded Aggregation (Conditional)}

We consider AC$^0$ augmented with bounded aggregation operators
(e.g., threshold gates with fixed thresholds).

\begin{conjecture}
Bounded-aggregation update systems admit bounded-rank refinement
simulation with polylogarithmic EntropyDepth distortion.
\end{conjecture}

This conjecture, if proved, would extend the unconditional frontier.
At present it remains open and is not used elsewhere.
