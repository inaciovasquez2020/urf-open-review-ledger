# URF-SG Theorem: Cycle Graph Rigidity

Let C_n be the cycle graph with Laplacian L.

Let V be any admissible perturbation in the URF-SG sense,
i.e. any D_n-equivariant operator.

Then for any consecutive isolated spectral gap
(E_k, E_{k+1}) with k ≠ 0 and (if n even) k ≠ n/2:

Π V Π = 0

Therefore:

|γ_k(ε) − γ_k(0)| = O(ε^2)

for all sufficiently small ε.

This is a representation-theoretic rigidity theorem,
not a perturbation argument.
