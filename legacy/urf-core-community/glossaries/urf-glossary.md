# URF Glossary

## URF
Unified Rigidity Framework.

## Canonical repository
The authoritative repository whose statements define the official line.

## Community repository
The additive repository for examples, exposition, tests, implementations, and explicitly non-canonical extensions.

## Canonical boundary
The rule separating authoritative claims from additive public material.

## Promotion
A maintainer-controlled transfer from the community layer into the canonical layer.

## Additive contribution
A contribution that extends usability, legibility, or verification without modifying canonical authority.
