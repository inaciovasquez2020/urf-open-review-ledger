\begin{remark}[External validation (machine-adversarial)]
An independent adversarial counterexample search (Gemini) did not refute
Theorem~\ref{thm:urf-core-lg-btw} in the bounded-treewidth regime, under the
dependency $k\ge f(t)$ and with $\mathsf{CycRank}_{R}(G)$ defined as the
$\mathbb F_2$-rank of the subspace generated by cycles of length $\le 2R$.
\end{remark}
