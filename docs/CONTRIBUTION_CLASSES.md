# Contribution Classes

## A. Examples
Worked instances of public definitions, interfaces, or theorems.

## B. Exposition
Readable maps, glossaries, dependency outlines, educational rewrites.

## C. Tests
Regression checks, witness checks, CI checks, executable validation.

## D. Implementations
Reference code, utilities, renderers, analyzers, adapters.

## E. Visualizations
Diagrams, graphs, structure maps, interactive teaching tools.

## F. Non-canonical extensions
New additive material explicitly marked non-canonical.
