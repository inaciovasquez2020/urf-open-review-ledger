Title: Replacement of FO^k by Bounded-Rank Logic

Status: Adopted restriction

Change:
All normalization and locality claims are stated using bounded-rank
logic (BRL) instead of FO^k.

Rationale:
1. BRL subsumes FO^k while remaining strictly local.
2. BRL is stable under refinement simulations of bounded-fan-in
   update systems.
3. BRL avoids dependence on variable-count artifacts.

Scope:
Every occurrence of FO^k in normalization statements is to be read
as “bounded-rank logical refinement”.

Effect:
No existing negative results are weakened.
All positive normalization results are preserved where stated.
