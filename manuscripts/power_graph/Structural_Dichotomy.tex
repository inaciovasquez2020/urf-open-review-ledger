\section{Structural Dichotomy for Local Refinement Depth}

For bounded-degree families with fixed $R$:

\[
\chi(G_n^{(2R)}) = O(1)
\]

Thus covering depth is constant.

If instead diameter satisfies

\[
\operatorname{diam}(G_n) \le 2R,
\]

then

\[
\chi(G_n^{(2R)}) = n,
\]

and linear depth is forced.

\paragraph{Conclusion.}
Local bounded-degree systems admit constant refinement depth.
Linear entropy depth requires structural collapse of the metric geometry.
