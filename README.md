URF Open Review Ledger

This repository is the public registry and review ledger for URF manuscripts.

Manuscripts themselves live in separate repositories.
This repository records:
- manuscript identifiers
- immutable version pointers (repo, tag, commit, hash)
- public review records
- review state transitions
- governance and acceptability rules
