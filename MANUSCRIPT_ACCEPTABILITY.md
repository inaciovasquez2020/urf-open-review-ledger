Manuscript Acceptability Standard (URF)

A manuscript is admissible for this ledger only if it satisfies the following.

Required sections, in order:

1. Title page
   Title, author(s), date, version identifier.

2. Abstract
   Clear scope.
   Explicit claim type: theorem, framework, negative result, or conditional result.

3. Scope and non-claims
   Explicit statement of what is not being claimed.

4. Definitions
   All nonstandard terms defined.
   Symbols fixed once.

5. Main statements
   All theorems, propositions, axioms explicitly labeled.
   Conditional statements explicitly marked.

6. Proofs or arguments
   Proofs reference only prior definitions and statements.
   Any gap must be stated as a precise missing lemma or assumption.

7. Dependencies
   External results listed explicitly and classified as:
   standard, literature-dependent, conjectural, or conditional.

8. Limitations
   Known failure modes and boundaries of applicability.

9. References
   Sufficient to locate each cited result.

Reference rules:

- Allowed: journals, preprints, DOI-tagged reports, versioned repositories, standards.
- Software references must include author, repository, version or commit, year, and URL.
- No anonymous blogs.
- No unverifiable consensus claims.

A frozen version (tag + commit) is required before review begins.
