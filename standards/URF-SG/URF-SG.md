# URF-SG — Spectral Gap Rigidity Standard

Status: Conditional / Standards Object

Invariant:
There exists α > 0 such that for all f ⟂ ker(Per):

⟨f, H_Φ f⟩ ≥ α ‖f‖²

Finite reduction:
Let ℓ₁,…,ℓ_m be linear observables.
Define Gram matrix G_ij = ⟨∇ℓ_i, ∇ℓ_j⟩.

Certificate condition:
λ_min(G) > 0.
